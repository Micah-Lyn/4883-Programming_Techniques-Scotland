612 - DNA Sorting
### Micah-Lyn Scotland
### Description:

This program catalogues a sequence of DNA strings (sequences containing only the four
letters A, C, G, and T) in
order of “sortedness”, from “most sorted” to “least sorted”. All the strings are of the same length.
